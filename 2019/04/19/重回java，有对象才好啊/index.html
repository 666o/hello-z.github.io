借用知乎上的一段话

java特性

耳熟能详

封装、继承、多态

封装：是指隐藏对象的属性和实现细节，仅对外提供公共访问方式。

可以将变化隔离、提高重用性、增强安全性

继承和多态就一起说了

私以为多态是依附在继承的特性上的

继承解决了代码重用问题，

然后声明某个子类兼容于某基类，外部调用者可无需关注其差别

而多态的定义为：基于对象所属类的不同，外部对同一个方法的调用，实际执行的逻辑不同。

很显然，多态实质上是继承的实现细节
