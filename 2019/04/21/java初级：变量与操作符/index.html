java初级：变量与操作符

发表于 2019-04-21 更新于 2020-10-03 分类于 java 阅读次数：

如题

变量

定义

变量的定义是： 用来命名一个数据的标识符

基本类型

一个变量的类型，决定了该变量可以包含什么样的值。
Java中有八种基本类型，都是Java语言预先定义好的，并且是关键字。

整型
byte 8位 -128~127

short 16位 -32768~32767

int 32位

long 64位

字符型
char 16位 只能存放一个字节

注意： String不是基本类型，而是一个类

浮点型
float 32位

double 64位

注意： 默认的小数值是double类型的

如果想声明为float类型，应当在数字后面加一个字母f

布尔型

true 1位

false 1位

字面值

创建一个Hero对象会用到new关键字

但是给一个基本类型变量赋值却不是用new

因为基本类型是Java语言里的一种内置的特殊数据类型

并不是某个类的对象

给基本类型的变量赋值的方式叫做 字面值

类型转换

不同类型之间的数据可以互相转换，但是要满足一定的规则

低精度向高精度转换时，都可以转

高精度向低精度转换时，需要强制转换，而且结果可能错误

基本数据类型的自动转换：

为什么4个字节的float排在8个字节的long后面？

因为他们在内存中的存储方式不同

float是用的 V=(-1)^s * M * 2^E

简单来说 float用32位的空间的表述范围比64位的Long要大，但是float某些精度上无法表示

但是不可能这么完美，所以在进行转换时，也存在精度丢失

表达式类型转换的自动提升规则：
　　　　1.所有的byte类型、short类型将被提升到int类型
　　　　2.整个算数表达式的数据类型自动提升到与表达式中最高等级操作数同样的类型。
所以两个int相除是得不到float或者是double的，只能是int

操作符

算术操作符

基本的有：

+ - * / % ++ --

关系操作符

> < >= <= == !=

返回的是布朗类型

逻辑操作符

& && | || ！ ^

短路与长路的区别在于：

长路两边都会被运算

短路只要能够判断了 ，就不会运算第二个表达式

长路是& and |

三元操作符

表达式?值1:值2

如果表达式为真 返回值1

如果表达式为假 返回值2
