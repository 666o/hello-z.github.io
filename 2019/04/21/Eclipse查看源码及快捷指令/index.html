Eclipse查看源码及快捷指令

发表于 2019-04-21 更新于 2020-10-03 分类于 java 阅读次数：

记录一次Eclipse使用时遇到的查看源码的问题

顺便写点Eclipse中的一些快捷指令

源码

查看源码

简单的有两种操作

可以直接选中方法然后右键选择声明

也可以选中方法然后按下F3

BUT 我遇到了一个问题

由于我已经解决了这个问题，所以图是盗的

解决方法

其实就是Eclipse找不到src.zip文件

只要设置一下就行了

首先从你Eclipse的窗口（windows）进入首选项

然后选中Java中的 已安装的JRE

编辑第二条的源代码链接（我的是 “无”）

点击源代码附件，选择外部文件

指向你的JDK目录下的src.zip

问题解决

快捷指令

批量修改

ALT+SHIFT+R

快速输入主方法

  1. 敲入main

  2. alt+/

快速输入System.out.println

  1. 敲入syso

  2. alt+/

快速助手

ctrl+shift+o

import 类

去掉未使用的类 等等都可以使用

快速注释

选中一行或者多行代码 ctrl+shift+c

toString getter setter 构造方法的快速生成

  1. 通过菜单-> Source 就可以看到快速生成的选项

  2. 直接使用快捷键 shift+alt+s也可以弹出这个选项
