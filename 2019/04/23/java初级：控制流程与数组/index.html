switch可以使用byte,short,int,char,String,enum

不支持浮点型以及long

因为浮点型不好直接比较

而Long类型是因为 switch 是比较int

向下转型会导致数据丢失

数组

定义：数组是一个固定长度的，包含了相同类型数据的容器

创建数组

int[]a or int a[]; 声明了一个数组变量

[]表示该变量是一个数组

int 表示数组里的每一个元素都是一个整数

a 是变量名

但是，仅仅是这一句声明，不会创建数组

创建数组的时候，要指明数组的长度

new int[5]

访问通过下标

长度通过属性.length得知

数组赋值

1.分配空间与赋值分步进行

2.分配空间，同时赋值

增强型for

个人感觉类似python的 for num in nums

例：

等于通过迭代来获取数组中的值，所以不能修改数组，只能取值

复制数组

数组的长度是不可变的，一旦分配好空间，是多长，就多长，不能增加也不能减少

java提供 arraycopy 方法复制数组

System.arraycopy(src, srcPos, dest, destPos, length)

src: 源数组
srcPos: 从源数组复制数据的起始位置
dest: 目标数组
destPos: 复制到目标数组的起始位置
length: 复制的长度

二维数组

二维数组，里面的每一个元素，都是一个一维数组

所以二维数组又叫数组的数组

数组的工具类

Arrays 是针对数组的工具类，可以进行 排序，查找，复制填充等功能。

Arrays提供了一个copyOfRange方法进行数组复制。
// copyOfRange(int[] original, int from, int to)
// 第一个参数表示源数组
// 第二个参数表示开始位置(取得到)
// 第三个参数表示结束位置(取不到)

Arrays提供了一个toString()方法，直接把一个数组，转换为字符串

sort方法

底层实现：当长度小于47时，使用插入排序

长度在47-286之间时，使用快排

大于286时，如果不是基本类型，就用归并排序

binarySearch进行查找

But 得先排序，因为这个方法用的二分查找实现的

Arrays.equals，判断是否相同
