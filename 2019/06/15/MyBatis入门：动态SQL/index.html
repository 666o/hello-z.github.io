概述

MyBatis的动态SQL的几种常用元素

元素 作用 备注
if 判断语句 单条件分支判断
choose(when,otherwise) 相当于Java中的switch和case语句 多条件分支判断
foreach 循环语句 在in语句等列举条件时常用
trim(where,set) 辅助元素，用于处理特定的SQL拼装问题，比如去掉多余的and/or等 用于处理SQL拼装问题

if元素
